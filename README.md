# SPICE-MNA

Repositório de estudo e documentação sobre **Modified Nodal Analysis (MNA)** com foco em simuladores do tipo **SPICE**.

O projeto foi estruturado em **Quarto** para funcionar como material técnico incremental, com ênfase em:

- fundamentos de MNA;
- formulação matricial;
- regras de stamping;
- exemplos resolvidos;
- aspectos numéricos relevantes para simuladores.

## Visualização do site

O site publicado pode ser acessado em:

[https://haulisson.github.io/SPICE-MNA/](https://haulisson.github.io/SPICE-MNA/)

## Estrutura principal

- `quarto/`: conteúdo do site em Quarto
- `quarto/mna/`: páginas técnicas sobre MNA
- `docs/`: documentação auxiliar do projeto
- `upstream/`: referências locais fora do controle de versão

## Publicação

A publicação do site é feita com **GitHub Pages** via **GitHub Actions**.
