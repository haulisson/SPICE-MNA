# upstream/

Esta pasta existe para armazenar materiais de referência locais que apoiam o estudo e a escrita do conteúdo do repositório.

## Regra de uso

Os arquivos dentro de `upstream/` **não fazem parte do controle de versão**.

Isso é intencional:

- neste computador você pode manter um conjunto de referências;
- em outro computador você pode manter arquivos diferentes;
- o conteúdo pode variar entre ambientes sem afetar o repositório.

## Exemplos de uso

Você pode colocar aqui:

- notas pessoais;
- PDFs;
- artigos;
- imagens de apoio;
- rascunhos temporários;
- documentos exportados de outras ferramentas.

## Restrições

- não tratar `upstream/` como fonte oficial do projeto;
- não copiar código externo diretamente para o repositório;
- usar essa pasta apenas como apoio local e leitura auxiliar.

## Versionamento

O repositório mantém apenas este `README.md` para documentar a finalidade da pasta.
